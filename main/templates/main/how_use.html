Полезная информация

  1. Для начала Вам необходимо загрузить хотя бы один файл расширения .py (нажмите «Загрузите новый файл»).

  2. Загруженный вами файл сразу же появится в левом меню.

    -- Обратите внимание, при загрузке существующего файла его содержимое перезаписывается, а его версия увеличивается на 1

  3. Теперь у Вас есть возможность перейти к этому файлу для проведения тестов.

  4. На странице тестов вы найдете информацию о файле, такую ​​как: имя, версия, выполненные тесты и ошибки в них.

  5. Также внутри одного теста вы можете просмотреть информацию о различных ошибках, нажав соответствующую кнопку

  6. Во вкладке отчетов вы можете скачать отчеты для тех файлов, где тесты проводились хотя бы один раз.
